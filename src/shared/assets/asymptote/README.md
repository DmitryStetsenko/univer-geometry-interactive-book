# Asymptote Графіка та Вихідні Коди

Ця папка містить вихідні коди геометричних креслень у форматі Asymptote (`.asy`) та їх скомпільовані векторні версії (`.svg`), які використовуються в інтерактивному підручнику.

## Опис файлів у папці

### 1. Теорема Піфагора (Покрокове доведення перегрупуванням)
*   **`pythagoras_step1.asy` / `pythagoras_step1.svg`**: Крок 1. Геометричний зміст. Малює прямокутний трикутник з катетами $a=3, b=4$ та квадратами, побудованими на його сторонах (червоний площею 9, синій — 16, зелений — 25).
*   **`pythagoras_step2.asy` / `pythagoras_step2.svg`**: Крок 2. Перше розташування. Малює великий квадрат $(a+b) \times (a+b)$ із чотирма трикутниками у кутах та центральним зеленим квадратом площею $c^2 = 25$.
*   **`pythagoras_step3.asy` / `pythagoras_step3.svg`**: Крок 3. Другорядне розташування. Малює перегруповані прямокутники в кутах, звільняючи два квадрати площею $a^2 = 9$ та $b^2 = 16$.

### 2. Демонстрація можливостей Asymptote (Складні алгоритмічні креслення)
*   **`koch_snowflake.asy` / `koch_snowflake.svg`**: Математичний рекурсивний фрактал «Сніжинка Коха» з глибиною рекурсії 4. Показує роботу з векторами, рекурсією та побудовою замкнутих векторних контурів.
*   **`spirograph.asy` / `spirograph.svg`**: Інтерактивний спірограф (крива гіпотрохоїди). Відображає три взаємопов'язані математичні орнаменти, згенеровані в циклі через параметричні рівняння з плавними переходами кольорів.
*   **`lorenz_attractor.asy` / `lorenz_attractor.svg`**: Хаотичний атрактор Лоренца (теорія хаосу). Відображає обчислену в циклі тривимірну траєкторію диференціальних рівнянь, спроєктовану в двовимірний векторний простір, з градієнтною розмальовкою перебігу часу.

## Інструкція з компіляції

Для перетворення `.asy` файлів у формат `.svg` використовується компілятор Asymptote (версія 3.0+):

```bash
# Компіляція конкретного файлу
asy -f svg <ім'я_файлу>.asy

# Приклад
asy -f svg koch_snowflake.asy
```

Примітка: Компілятор Asymptote використовує систему LaTeX (наприклад, MiKTeX чи TeX Live) для рендерингу математичних формул і написів. Сгенеровані `.svg` файли автоматично додаються у збірку Vite та відображаються у веб-інтерфейсі.
